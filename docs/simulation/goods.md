## OpenVic2 Structure

```mermaid
classDiagram
    class GoodCategory {
        <<enumeration>>
        Military
        Raw
        Consumer
        Industrial
    }

    class Good {
        String id
        GoodCategory category
        Fixed32/Float32 cost
        Colour colour
        Boolean isAvailableAtStart
        Boolean isTradeable
        Boolean isMoney
        Boolean hasOverseasPenalty
    }

    Good o-- GoodCategory
```
